# Changelog

## v1 - Initial release

- Included signals:
  - Block
  - Path
  - One-way Path
  - No-entry
- Included styles:
  - Semaphores
  - Searchlight
  - Color lights
  - B&O Color-position
- Other features
  - Ability to replace default graphics
  - Ability to use double-headed block signals instead
  - Ability to set signal styles individually per signal GUI row (when replacing default graphics only)
